# aski
Elinde kullanmadıkları fazla eşyası olan insanların onları ihtiyaç sahibi insanlarla "ücretsiz bir şekilde" paylaşabildiği bir mobil uygulama. İnsanlar vermek istedikleri fazla ürünün kategorisini girip, fotoğrafını çekip uygulamaya yükleyebilecekler, ihtiyacı olan insanlar da bir ürünü seçip sahibiyle iletişime geçip ürünü alabilecek.
